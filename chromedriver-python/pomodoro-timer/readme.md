
## 该应用包含以下特性：

1. 三个核心文件：

   - `index.html` - 应用的主页面结构
   - `style.css` - 响应式样式设计
   - `script.js` - 完整的JavaScript逻辑实现

2. 功能特点：

   - 三种模式：专注时间（25分钟）、短休息（5分钟）、长休息（15分钟）
   - 开始、暂停、重置控制按钮
   - 自动循环模式切换（专注→短休息→专注→短休息→专注→长休息）
   - 番茄钟计数器
   - 响应式设计，适配移动设备
   - 提示音和浏览器通知功能
   - 现代化的用户界面设计

3. 技术实现：

   - 使用原生JavaScript，无需外部依赖
   - 面向对象编程结构
   - Web Audio API实现提示音
   - HTML5 Notifications API实现桌面通知
   - CSS3动画和过渡效果
