# 桌面截图工具 (Snip Tool)

一个简单易用的桌面截图工具，支持悬浮图标、区域选择、图像编辑和保存功能。

## 功能特性

- **悬浮图标**：桌面右下角显示可拖动的悬浮图标
- **区域截图**：点击图标后，拖动鼠标选择要截图的区域
- **图像编辑**：
  - 画笔工具：自由绘制
  - 矩形工具：绘制矩形框
  - 文字工具：添加文字标注
- **自定义选项**：
  - 多种颜色选择（红、蓝、绿、黑、黄）
  - 可调节笔刷大小（1-20）
  - 撤销操作
  - 清空所有标注
- **保存功能**：支持 PNG 和 JPEG 格式，自定义保存路径
- **重新截图**：可随时重新截图

## 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

1. 运行程序：
```bash
python main.py
```

2. 程序启动后，会在屏幕右下角显示一个蓝色的剪刀图标

3. **进行截图**：
   - 点击悬浮图标
   - 屏幕会变暗，显示全屏截图窗口
   - 拖动鼠标选择要截图的区域（会显示红色边框）
   - 释放鼠标完成截图
   - 按 ESC 键可取消截图

4. **编辑截图**：
   - 在编辑器中选择工具（画笔、矩形、文字）
   - 选择颜色和笔刷大小
   - 在截图上进行标注和编辑
   - 可以使用"撤销"和"清空"按钮管理标注

5. **保存截图**：
   - 点击"保存"按钮
   - 选择保存位置和文件名
   - 选择文件格式（PNG 或 JPEG）
   - 确认保存

6. **重新截图**：
   - 点击"重新截图"按钮可以重新选择截图区域

## 快捷操作

- **拖动图标**：按住悬浮图标可以拖动到任意位置
- **ESC 键**：在截图选择时按 ESC 可以取消截图

## 系统要求

- Python 3.7+
- Pillow 库（用于图像处理）
- 主要支持 Windows 系统（其他系统可能需要调整）

## 技术说明

- 使用 Tkinter 构建 GUI 界面
- 使用 PIL/Pillow 进行图像处理
- 支持跨平台（主要针对 Windows 优化）

## 注意事项

1. 截图时会暂时隐藏悬浮图标，保存或取消后会重新显示
2. 编辑器的标注当前只保存在画布上，保存为文件时可能不会包含标注内容
3. 如遇到截图失败，请检查是否有足够的屏幕权限
